## üöÄ Installation & Setup

### Prerequisites
- Node.js 18+ and npm
- Python 3.9+
- PostgreSQL 14+
- Git

### Step 1: Clone & Navigate
```bash
# Already in VS Code terminal
pwd  # Confirm you're in brands-galaxy folder
```

### Step 2: Frontend Setup
```bash
# Create Next.js app
npx create-next-app@latest frontend

# Choose these options:
# Would you like to use TypeScript? No
# Would you like to use ESLint? Yes
# Would you like to use Tailwind CSS? Yes
# Would you like to use `src/` directory? Yes
# Would you like to use App Router? Yes
# Would you like to customize the default import alias? No

# Navigate and install dependencies
cd frontend
npm install axios zustand react-hot-toast @stripe/stripe-js lucide-react
```

### Step 3: Backend Setup
```bash
# Go back to root
cd ..

# Create backend folder
mkdir backend
cd backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# Install Python packages
pip install fastapi uvicorn sqlalchemy psycopg2-binary python-dotenv pydantic pydantic-settings stripe python-multipart pillow python-jose passlib bcrypt

# Save requirements
pip freeze > requirements.txt
```

### Step 4: Database Setup
```bash
# Create PostgreSQL database
psql -U postgres
CREATE DATABASE brands_galaxy;
\q
```

### Step 5: Environment Variables

Create `backend/.env`:
```env
DATABASE_URL=postgresql://postgres:password@localhost:5432/brands_galaxy
SECRET_KEY=your-secret-key-here-change-this
STRIPE_SECRET_KEY=sk_test_your_stripe_key
STRIPE_PUBLISHABLE_KEY=pk_test_your_stripe_key
ACCESS_TOKEN_EXPIRE_MINUTES=30
```

Create `frontend/.env.local`:
```env
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_your_stripe_key
```

## üì¶ Database Schema

### Users Table
- id (UUID, Primary Key)
- email (String, Unique)
- password_hash (String)
- full_name (String)
- phone (String, Optional)
- address (JSON)
- is_admin (Boolean, Default: False)
- created_at (DateTime)
- updated_at (DateTime)

### Products Table
- id (UUID, Primary Key)
- name (String)
- slug (String, Unique)
- description (Text)
- price (Decimal)
- discount_price (Decimal, Optional)
- brand (String)
- category_id (Foreign Key)
- images (JSON Array)
- stock_quantity (Integer)
- is_featured (Boolean)
- created_at (DateTime)
- updated_at (DateTime)

### Categories Table
- id (UUID, Primary Key)
- name (String)
- slug (String, Unique)
- image (String, Optional)

### Orders Table
- id (UUID, Primary Key)
- user_id (Foreign Key)
- order_number (String, Unique)
- total_amount (Decimal)
- status (Enum: pending, paid, shipped, delivered, cancelled)
- payment_method (String)
- stripe_payment_id (String, Optional)
- shipping_address (JSON)
- created_at (DateTime)
- updated_at (DateTime)

### Order Items Table
- id (UUID, Primary Key)
- order_id (Foreign Key)
- product_id (Foreign Key)
- quantity (Integer)
- price (Decimal)

## üîê API Endpoints

### Authentication
- POST `/api/auth/register` - Register new user
- POST `/api/auth/login` - Login user
- GET `/api/auth/me` - Get current user

### Products
- GET `/api/products` - Get all products (with filters)
- GET `/api/products/{id}` - Get single product
- POST `/api/products` - Create product (Admin)
- PUT `/api/products/{id}` - Update product (Admin)
- DELETE `/api/products/{id}` - Delete product (Admin)

### Cart & Orders
- POST `/api/orders` - Create order
- GET `/api/orders` - Get user orders
- GET `/api/orders/{id}` - Get order details
- POST `/api/payments/create-intent` - Create Stripe payment intent

### Admin
- GET `/api/admin/dashboard` - Dashboard stats
- GET `/api/admin/orders` - All orders
- PUT `/api/admin/orders/{id}/status` - Update order status
